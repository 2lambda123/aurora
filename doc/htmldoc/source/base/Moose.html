      Moose.C

      The Moose.C file is an core framework file containing several useful global variables, task list, utilities, typedefs, and other core-internal code. One of the most commonly consulted blocks of code in Moose.C is the task list ordering. This list dictates the task order in which MOOSE assembles complete simulations.
