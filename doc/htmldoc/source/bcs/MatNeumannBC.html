      MatNeumannBC

      Imposes the integrated boundary condition , where is a constant, is a material property, and is a coefficient defined by the kernel for .

      Description

      MatNeumannBC is a generalization of NeumannBC which is used for imposing flux boundary conditions on systems of partial differential equations (PDEs) where the flux is represented by the product of a constant and a MOOSE Material Material. That is, for a PDE of the form where is the domain, and is its boundary, a MatNeumannBC object can be used to impose condition (3) if the material is well-defined for all relevant times and . In this case, the boundary_material parameter corresponds to a MOOSE Material object, and the user must define one or more sidesets corresponding to the boundary subset via the boundary parameter.

      Example Input Syntax

  type = MatNeumannBC
  variable = u
  boundary = top
  value = 2
  boundary_material = hm
