      HFEMDirichletBC

      Imposes the Dirichlet BC with HFEM.

      Description

      HFEMDirichletBC is used for imposing the Dirichlet boundary with HFEM (hybrid finite element method). A HFEMDirichletBC object is for on the subset of the boundary denoted by . Thus it requires the "variable" parameter for , the "lowerd_variable" parameter for and the "value" corresponds to . Refer to DGKernels System for the full HFEM weak form for Poisson's equation.

      Example Input Syntax

        [all]
    type = HFEMDirichletBC
    boundary = 'left right top bottom'
    variable = u
    lowerd_variable = lambdab
    uhat = uhat
  []
