Detailed Description

Transfer a pointer to the FEProblem from the parent to child app in a MultiApp.

Specifically, pass the data into the MoabUserObject with the name contained in moabname, and fail if this does not exist. Also perform some intialisation of the MoabUserObject that should only be done once.
